1. การเตรียมข้อมูล (Data Preparation)

โปรเจกต์นี้ใช้ข้อมูลสองชุดหลัก ได้แก่ House Price Dataset และ MNIST Handwritten Digits Dataset

House Price Dataset: เป็นข้อมูลเกี่ยวกับราคาบ้านโดยมีฟีเจอร์ต่างๆ เช่น พื้นที่บ้าน (area), จำนวนห้องนอน (bedrooms), จำนวนห้องน้ำ (bathrooms), จำนวนชั้น (stories) และที่จอดรถ (parking)

MNIST Dataset: เป็นชุดข้อมูลภาพขนาด 28x28 pixel ที่เป็นตัวเลข 0-9 ซึ่งใช้สำหรับการจำแนกประเภทด้วย Neural Network

ก่อนนำข้อมูลมาใช้ เราทำการล้างข้อมูล (Data Cleaning) โดยการลบค่า Missing Values และจัดรูปแบบข้อมูลให้เหมาะสม รวมถึงการแบ่งข้อมูลออกเป็นชุด Train-Test โดยใช้ `train_test_split`

2. ทฤษฎีของอัลกอริทึมที่ใช้

2.1 Machine Learning: Random Forest

Random Forest เป็นอัลกอริทึมแบบ Ensemble Learning ที่ใช้หลาย Decision Trees เพื่อเพิ่มความแม่นยำและลด Overfitting

การทำงานของ Random Forest คือการสร้างหลายๆ ต้นไม้และนำผลลัพธ์มาคำนวณค่าเฉลี่ยเพื่อให้ได้ผลลัพธ์ที่ดีที่สุด

2.2 Neural Network: Convolutional Neural Network (CNN)

CNN เป็นอัลกอริทึมที่เหมาะสำหรับการวิเคราะห์ภาพ โดยใช้ Convolutional Layers เพื่อตรวจจับ Feature สำคัญจากภาพ

ในโมเดลของเราใช้เลเยอร์ Conv2D, MaxPooling2D, Flatten, และ Dense Layers เพื่อสร้างโครงสร้างที่สามารถจำแนกตัวเลขจาก MNIST ได้อย่างแม่นยำ
