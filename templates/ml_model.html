ขั้นตอนการพัฒนาโมเดล

Train Machine Learning Model

1. โหลดข้อมูล House Price จาก CSV (`train.csv`)

2. ล้างข้อมูล (Data Cleaning) และลบค่าที่ขาดหาย

3. เลือกฟีเจอร์ที่ใช้ (`area, bedrooms, bathrooms, stories, parking`)

4. แบ่งข้อมูลเป็น Train-Test (80%-20%)

5. Train โมเดล `RandomForestRegressor(n_estimators=100)`

6. ทดสอบโมเดล และประเมินผลด้วย Mean Absolute Error (MAE)

7. บันทึกโมเดลเป็นไฟล์ `house_price_model.sav`

Train Neural Network Model

1. โหลดข้อมูล MNIST จาก CSV (`mnist_train.csv`)

2. แปลงภาพเป็น NumPy Array และ Normalize เป็น 0-1

3. One-hot encode labels (`to_categorical()`)

4. แบ่งข้อมูลเป็น Train-Test (80%-20%)

5. สร้างโมเดล CNN ที่มี Conv2D, MaxPooling2D, Flatten, และ Dense Layers

6. Train โมเดล `epochs=10, batch_size=32`

7. ทดสอบ Accuracy บน Test Set

8. บันทึกโมเดลเป็น `mnist_model.h5`
